# plaso-open-specs
Plaso Mobile Service CocoaPods specs.

Plaso 移动服务 iOS CocoaPods 仓库。


## 使用
- Podfile中指定仓库位置，

```
# Master仓库不要遗漏
source 'https://github.com/CocoaPods/Specs.git'
source  'https://github.com/PlasoOpenPlatform/plaso-open-specs.git'

```
